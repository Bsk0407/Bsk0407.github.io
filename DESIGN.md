# Portfolio Design Theme

## Theme

Clean AI research portfolio

## Direction

- Use a calm, readable layout for a student portfolio.
- Keep the first screen focused on the name, AI interests, and profile image.
- Use clear sections: About Me, Interests, Projects, Contact.
- Avoid heavy decoration so the content is easy to scan.

## Visual Style

- Background: light gray-blue
- Main surface: white
- Accent color: teal
- Secondary highlight: warm amber
- Card radius: 8px
- Typography: simple sans-serif with strong headings

## Content Priorities

1. Show 김범수 as the owner of the page.
2. Present interests in LLM and multimodal AI.
3. Include a project section even if projects are not available yet.
4. Make contact information easy to find.
